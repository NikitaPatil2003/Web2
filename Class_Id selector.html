___CLASS AND ID SELECTOR___



FORMATTING TAGS

  • P TAG:- This is the paragraph tag
  • U TAG:- This is the underline tag
  • I TAG:- This is the italic tag
  • SUP TAG:- This is the supescript tag
  • SUB TAG:- This is the subscript tag

LIST TAGS

  1. UL TAG:-This is the unorder list tag
  2. OL TAG:-This is the order list tag
  3. DL TAG:-This is the defination list tag
  4. SELECT TAG:- This is the dropdown list tag
